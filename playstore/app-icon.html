Rappel export : sur Android API 26+ (la quasi-totalité du parc) l'icône est rendue depuis les vector drawables — déjà à jour. Les PNG mipmap-*dpi ne servent que sur API 24–25 ; icon-512.png et la feature graphic sont aussi des PNG figés. Tous se régénèrent depuis cette source (Android Studio → Image Asset, ou rasterisation de icon-master.svg).
